Reno System Dynamics (reno-sd)
==============================

Welcome to Reno! Reno is a library for building, running, and analyzing `system
dynamics models <https://en.wikipedia.org/wiki/System_dynamics>`__ in Python.

.. figure:: _static/explorer_screenshot.png
   :align: center

Some of the features of this library include:

* PyMC integration for running Bayesian inference on models with prior
  probability distributions
* Customizable stock and flow diagram generation
* Plotting functions to quickly create model visualizations
* Save and load models to/from JSON
* Web-based model exploration UI
